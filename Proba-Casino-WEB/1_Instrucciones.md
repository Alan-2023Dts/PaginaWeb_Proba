Crea una página web interactiva que demuestre por qué un mito es falso o engañoso. La página debe cumplir los siguientes requisitos:

Estructura y funcionalidad:

La web debe incluir 8 minijuegos o simulaciones visuales, cada uno en una página diferente.
Debe haber un menú de navegación que permita ir a cada uno de los 8 juegos fácilmente.
La web debe subirse a GitHub Pages para que sea accesible en línea.

Tecnologías sugeridas:

HTML
CSS (con animaciones si es necesario)
JavaScript
Canvas o animaciones CSS
Chart.js opcional para visualizaciones

Requisitos de los juegos:

Cada juego debe incluir animación o simulación visual dinámica.
Debe haber interacción con el usuario, estilo casino (ej. clics, arrastres, selección de opciones, lanzamiento de dados, ruedas de azar).
Cada juego debe mostrar el mito que se está evaluando con una explicación clara.
El juego debe demostrar, mediante la simulación o la interacción, por qué el mito es falso o engañoso.

Extras opcionales:

Gráficos estadísticos con Chart.js para reforzar la demostración.
Feedback visual o sonoro que haga la experiencia más interactiva y atractiva.

Objetivo final:
Crear una experiencia educativa y entretenida donde los usuarios aprendan mediante la interacción que ciertos mitos son solo mitos, usando la lógica, la probabilidad o la evidencia visual.
